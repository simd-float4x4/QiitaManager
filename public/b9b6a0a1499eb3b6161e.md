# 課題

* 競技プログラミングをしているとこういうケースがあります。

> 任意の長さの配列Aと配列Bを合算した配列の中央値を求めなさい。


# 作業方針
* まずは配列Aと配列Bの合計の長さを求める。
* AとBの値を一つずつ大小比較していく
* 最後に中央値を取得する


* Javaなどであれば、

```java
int [] arr = new int[arr_a, arr_b];
```

のような形で記述することが可能なのですが、Swiftではこのような実装するにはどうしたらよいでしょうか。


# コード

* `Array.repating`を用いることが可能です。

```Swift
Array(repeating: 0, count: Int)
```
* あらかじめ、必要なメモリ領域を確保しておき、あとでそこから値を詰めていく方法を取ります。


# 使用例
```Swift
let ln1 = l1.count
let ln2 = l2.count
let array = Array(repeating: 0, count: ln1+ln2)
```
