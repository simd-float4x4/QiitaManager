### 発生したエラー
```Dart
class Cover {
  String author,

  Cover(this.author);
}
```
* クラス宣言したところで`author`は初期化されなければならないと怒られていた。

### 解決策

```Dart
class Cover {
  String author;  //「,」が間違い。「;」でフィールドを締める。

  Cover(this.author);
}
```

* typo. 文末が「;」ではなく「,」になっていたことが原因。

---

* これはこれとして`flutter_lint`が正常に動作していない気がする。
