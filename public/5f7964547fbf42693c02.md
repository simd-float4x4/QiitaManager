ドハマりしてしまいCRUDを検証するなどして大幅に時間ロスしたので備忘録。

# 発生した不具合
* `Project`と`Ideal`という２つのテーブルを持つアプリにて、一番最初に定義されている`Project`のデータのやり取りには問題が無いが、`Ideal`というデータのやり取りはなぜか全く保持されていない。

# 解決策
```diff_swift
let schema = Schema([
    Project.self,
+     Ideal.self,
])
```

# 解説
* スキーマ定義をしているファイルにて、`Schema`の中に新しく追加した`@Query`テーブル構造体を突っ込んでおく。（忘れない）
